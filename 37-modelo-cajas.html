Modelo de cajas

Cada elemento de la página tiene su propio cuadro, que se puede diseñar con CSS

Desde el interior hacia el exterior:

  • contenido
  • relleno
  • borde
  • margen

Fondo

Si se establece un fondo cubrirá a contenido y relleno.

En conjunto

  1. Se asigna a h1 un fondo rojo, algo de relleno y un borde negro.
  2. Se asigna a cada p un fondo azul, texto blanco, algo de relleno y un borde negro.
  3. Se asigna a cada elemento code un fondo blanco, texto negro, un poco de relleno y un borde delgado.

Ejercicio

  1. Añadir estilos a los h2s, ols, y lis de esta página.
  2. Revisar la sintaxis de margin para aumentar la separación de los distintos elementos de la página.
